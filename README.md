## 📁 Code Examples

All Java examples are provided in the `src/` directory with the following structure:
```
src/
├── basics/
├── oop/
├── exceptions/
├── collections/
├── memory/
└── advanced/
```

## 🛠️ How to Run Examples

1. Navigate to the specific example directory
2. Compile: `javac *.java`
3. Run: `java MainClass`
